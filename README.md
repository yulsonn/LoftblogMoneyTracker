# Описание проекта
LoftblogMoneyTracker - Android-приложение для управления личными финансами. 
Разработано в процессе обучения на курсе "Комплексное обучение разработке на Android" от LoftSchool.

# Технологии
* REST API: Retrofit
* ORM: ActiveAndroid
* Android Annotations
* Material Design conceptions
* G+ Authorization
